# TVF Mobile - Checklist de prise de captures

## Avant de capturer

- Ouvrir l'application dans Expo Go ou dans un build preview propre.
- Verifier que le logo TVF Mobile est net.
- Verifier que les textes ne sont pas coupes.
- Desactiver les notifications visibles sur le telephone.
- Utiliser une luminosite correcte et le mode clair.
- Ne pas afficher de données personnelles reelles.

## Sequence officielle

### 1. Accueil

But : montrer que l'utilisateur comprend en quelques secondes ce qu'il peut faire.

A verifier :

- logo visible ;
- boutons principaux lisibles ;
- aucun texte coupe ;
- navigation basse propre.

### 2. Signalement de lieu

But : montrer le coeur de l'application terrain.

A verifier :

- categorie visible ;
- adresse ou commune visible ;
- bouton photo/localisation visible ;
- formulaire pas trop dense.

### 3. Proposition de materiaux

But : montrer que TVF Mobile sert aussi au reemploi.

A verifier :

- type de materiau visible ;
- quantite visible ;
- etat et localisation visibles ;
- aucun mot coupe.

### 4. Proposition de bien

But : montrer le parcours proprietaire / bien dormant.

A verifier :

- type de bien visible ;
- objectif de la demande visible ;
- informations de pre-etude visibles.

### 5. Confirmation et ticket

But : rassurer l'utilisateur apres l'envoi.

A verifier :

- reference lisible ;
- statut clair ;
- message de confirmation compréhensible ;
- bouton de retour ou de suivi visible.

### 6. Mes demandes

But : montrer que l'utilisateur peut retrouver ses demandes.

A verifier :

- liste lisible ;
- statut visible ;
- date ou reference visible ;
- pas de vraie donnée personnelle.

### 7. Documents

But : montrer l'aide a la preparation du dossier.

A verifier :

- titres des pieces lisibles ;
- liste pas trop longue a l'ecran ;
- presentation claire.

### 8. Contact

But : montrer les canaux officiels TVF.

A verifier :

- e-mail officiel ;
- telephone officiel ;
- WhatsApp visible ;
- aucun numero personnel affiche.

## Controle final

Chaque capture doit etre validee avec ces points :

- lisibilité mobile ;
- contraste suffisant ;
- aucune faute visible ;
- aucune donnée sensible ;
- aucun texte coupe ;
- aucune image deformee ;
- rendu professionnel.
