# TVF Mobile - Textes associes aux captures

Ces textes peuvent accompagner les captures dans un dossier interne, une presentation ou une fiche store.

| Capture | Titre court | Texte d'accompagnement |
| --- | --- | --- |
| 01 | Choisir un parcours | Signaler un lieu, proposer des materiaux, presenter un bien ou rejoindre TVF depuis une interface simple. |
| 02 | Signaler sur le terrain | Localiser une situation, ajouter une description et joindre une photo lorsque c'est utile. |
| 03 | Valoriser les materiaux | Decrire un lot reutilisable pour permettre a TVF d'evaluer son potentiel de reemploi. |
| 04 | Proposer un bien dormant | Transmettre les premieres informations necessaires a l'etude d'un logement, local, batiment ou terrain. |
| 05 | Garder une reference | Chaque demande genere une reference de suivi pour faciliter les echanges avec TVF. |
| 06 | Retrouver ses demandes | L'utilisateur conserve localement l'historique des demandes transmises depuis son telephone. |
| 07 | Preparer les pieces | TVF Mobile aide a comprendre les documents utiles avant une instruction. |
| 08 | Contacter TVF | E-mail, telephone et WhatsApp permettent de joindre facilement Territoires Vivants France. |
