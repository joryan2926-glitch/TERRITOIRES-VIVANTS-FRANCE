# TVF Mobile - Dossier captures officielles

Ce dossier regroupe les captures officielles a produire pour Google Play, App Store ou une diffusion interne TVF.

## Regle de nommage

Enregistrer les fichiers dans `screenshots/` avec les noms suivants :

| Ordre | Nom de fichier attendu | Ecran a capturer | Objectif |
| --- | --- | --- | --- |
| 01 | `01-accueil-parcours.png` | Accueil TVF Mobile | Montrer les parcours principaux. |
| 02 | `02-signalement-lieu.png` | Formulaire Signaler un lieu | Montrer categorie, adresse, photo et localisation. |

## Formats recommandes

- Android : captures verticales 1080 x 1920 si possible.
- iOS : captures verticales iPhone, en gardant les marges de securite.
- Ne pas recadrer au point de couper la barre haute, les boutons ou la navigation basse.
- Utiliser le mode clair.

## Donnees de demonstration

Utiliser uniquement des donnees neutres :

- Commune : Saint-Etienne.
- Adresse : secteur centre-ville ou rue sans numero precis si necessaire.
- Contact : contact@territoiresvivantsfrance.fr.
- Telephone : 04 65 81 54 69.
- Reference : TVF-DEMO ou reference de test non personnelle.

Ne jamais afficher une vraie demande personnelle, une cle API, une adresse privee sensible ou une photo non autorisee.
