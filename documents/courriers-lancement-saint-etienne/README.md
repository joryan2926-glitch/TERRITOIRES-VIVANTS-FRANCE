# Kit courriers de lancement - TVF Saint-Etienne

Ce dossier contient les modeles de courriers prets a personnaliser pour contacter les interlocuteurs utiles au lancement de Territoires Vivants France a Saint-Etienne.

## Organisation

- `00-index-kit-courriers-lancement-saint-etienne.docx` : index et mode d'emploi.
- `00-kit-complet-courriers-lancement-saint-etienne.docx` : tous les courriers dans un seul document.
- `01-...` a `20-...` : courriers separes par besoin.

## Regle d'utilisation

Verifier le destinataire, adapter la demande, joindre uniquement les pieces utiles et ne jamais annoncer de partenariat sans accord ecrit.
