# Courriers prets a envoyer - TVF Saint-Etienne

Ce dossier contient une version formalisee des courriers de prise de contact. Les fichiers DOCX sont modifiables et les PDF sont prets a joindre a un email ou a imprimer.

## Dossiers

- `docx/` : courriers Word modifiables.
- `pdf/` : courriers PDF prets a envoyer.

## Point de vigilance

Verifier l'adresse exacte lorsque le courrier cible une entreprise, une fondation ou un service dont le destinataire n'est pas encore choisi. Ne jamais annoncer un partenariat sans accord ecrit.
