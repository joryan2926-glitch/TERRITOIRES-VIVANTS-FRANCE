# Criteres d'acceptation et de refus des materiaux

Document operationnel modifiable - Banque de materiaux TVF

---

## 1. Principe

La banque de materiaux TVF n'accepte pas automatiquement tous les dons ou surplus. Chaque ressource doit etre utile, sure, identifiable, transportable, stockable et affectable a un projet.

TVF peut refuser un materiau sans remettre en cause la bonne intention du contributeur.

---

## 2. Criteres d'acceptation

| Critere | Condition minimale |
|---|---|
| Utilite | Le materiau peut servir a un projet TVF ou associatif identifiable |
| Etat | Bon etat, reutilisable ou reparable sans risque majeur |
| Identification | Type, quantite, dimensions, origine et photos disponibles |
| Securite | Pas de risque apparent pour personnes, stockage ou manipulation |
| Transport | Retrait possible avec moyens proportionnes |
| Stockage | Volume compatible avec le local disponible |
| Delai | Date de retrait compatible avec l'organisation TVF |
| Tracabilite | Bordereau, fiche ou registre remplissable |

---

## 3. Refus automatique ou orientation externe

| Ressource | Motif |
|---|---|
| Materiaux souilles ou humides | Risque sanitaire, stockage difficile |
| Amiante ou suspicion d'amiante | Traitement specialise obligatoire |
| Produits dangereux, solvants, peintures anciennes non identifiees | Risque securite et reglementation |
| Electricite non qualifiee ou non conforme | Risque technique |
| Sanitaires tres abimes | Faible reutilisation, hygiene, stockage |
| Gravats, terre, dechets melanges | TVF n'est pas une dechetterie |
| Materiaux sans autorisation de retrait | Risque juridique |
| Volumes trop importants sans stockage | Faisabilite insuffisante |

---

## 4. Statuts dans le registre

| Statut | Signification |
|---|---|
| Propose | Le contributeur a signale une ressource |
| A verifier | Il manque photos, etat, quantite ou autorisation |
| Accepte | La ressource est utile et peut etre prise en charge |
| Refuse | La ressource n'est pas compatible |
| Oriente | Une autre solution ou filiere est plus adaptee |
| Stocke | La ressource est entree dans le local TVF |
| Reserve | Une affectation est identifiee |
| Affecte | La ressource a ete remise a un projet |
