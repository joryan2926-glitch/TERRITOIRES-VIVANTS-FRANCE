# Scénarios types et cas d'usage - TVF

Document de présentation modifiable - version de travail

Important : les exemples ci-dessous sont des scénarios types. Ils ne doivent pas être présentés comme des projets réalisés.

## 1. Scénario type - commerce vacant

### Situation initiale

Un local commercial est fermé depuis plusieurs mois dans une rue de centre-ville. La vitrine donne une impression de déclin et le propriétaire ne dispose pas encore d'un projet solide.

### Besoin public

- maintenir une activité en centre-ville ;
- éviter la dégradation de la rue ;
- proposer un usage temporaire ou pérenne ;
- attirer des porteurs de projet.

### Intervention TVF possible

1. Qualifier le local.
2. Identifier les contraintes : état, accès, sécurité, loyer, travaux.
3. Échanger avec propriétaire et collectivité.
4. Étudier plusieurs usages : boutique éphémère, atelier, association, tiers-lieu, commerce de proximité.
5. Rechercher matériaux, compétences ou financement.
6. Préparer une convention.
7. Suivre l'usage et documenter les résultats.

### Indicateurs possibles

- local qualifié ;
- convention signée ;
- durée d'usage ;
- acteurs mobilisés ;
- activité accueillie ;
- travaux ou aménagements réalisés ;
- retours des usagers.

## 2. Scénario type - logement vacant

### Situation initiale

Un logement ancien est vacant et nécessite des travaux. Le propriétaire hésite à engager une remise en état faute de moyens, de temps ou de visibilité.

### Besoin public

- remettre un logement en usage ;
- lutter contre la vacance ;
- éviter la dégradation ;
- répondre à un besoin local d'habitat.

### Intervention TVF possible

1. Décrire le logement.
2. Vérifier la propriété et l'accord d'étude.
3. Identifier les travaux apparents.
4. Étudier les usages possibles : logement temporaire, logement solidaire, habitat partagé, accueil associatif.
5. Rechercher les ressources : matériaux, compétences, aides possibles, partenaires.
6. Formaliser les responsabilités.
7. Préparer un suivi et une restitution.

### Points à verrouiller

- propriété ;
- assurances ;
- sécurité ;
- travaux ;
- durée ;
- usage autorisé ;
- conditions de restitution.

## 3. Scénario type - friche ou terrain délaissé

### Situation initiale

Un terrain ou une friche est inutilisé, peu entretenu et perçu comme un espace sans fonction.

### Besoin public

- améliorer le cadre de vie ;
- créer un usage local ;
- favoriser la biodiversité ;
- limiter l'artificialisation ;
- impliquer les habitants.

### Intervention TVF possible

1. Identifier le propriétaire ou gestionnaire.
2. Évaluer les contraintes : pollution, accès, sécurité, voisinage, réglementation.
3. Étudier les usages : jardin partagé, espace nourricier, lieu associatif, espace pédagogique, événement temporaire.
4. Mobiliser habitants, associations, collectivité et entreprises.
5. Formaliser une convention d'usage.
6. Suivre l'entretien, les usages et les bénéfices.

### Indicateurs possibles

- surface qualifiée ;
- usage créé ;
- participants mobilisés ;
- actions réalisées ;
- contraintes levées ;
- retour des habitants.

## 4. Scénario type - matériaux de réemploi

### Situation initiale

Une entreprise dispose de matériaux issus d'un chantier, d'un changement de mobilier ou d'un stock dormant.

### Besoin public

- éviter le gaspillage ;
- réduire les coûts de certains projets ;
- soutenir l'économie circulaire ;
- alimenter des projets associatifs ou territoriaux.

### Intervention TVF possible

1. Identifier la ressource.
2. Décrire l'état, la quantité et la localisation.
3. Vérifier les conditions de récupération.
4. Déterminer si la ressource est réemployable.
5. Affecter à un projet validé.
6. Tracer la remise, le stockage et l'utilisation.

### Points à verrouiller

- propriété de la ressource ;
- état ;
- sécurité ;
- transport ;
- stockage ;
- affectation ;
- preuve d'utilisation.

## 5. Scénario type - bien solidaire à usage partagé

### Situation initiale

Un propriétaire possède un bâtiment ou un logement vacant qui nécessite des travaux. Il souhaite conserver la propriété mais accepterait un usage temporaire utile au territoire.

### Besoin public

- créer un logement, un local associatif, un atelier, un lieu de formation ou un service local ;
- valoriser un patrimoine inutilisé ;
- mobiliser matériaux et compétences ;
- expérimenter une coopération territoriale.

### Intervention TVF possible

1. Étudier le bien.
2. Vérifier les contraintes.
3. Comparer les scénarios d'usage.
4. Évaluer les besoins de travaux.
5. Rechercher matériaux, compétences, financement ou partenaires.
6. Définir une convention d'utilisation temporaire.
7. Préparer la restitution du bien en fin de période.

### Modalités possibles

- rénovation contre usage temporaire ;
- rénovation contre loyer solidaire ;
- rénovation avec partage de revenus selon convention ;
- mise à disposition encadrée pour un usage d'intérêt territorial.

## 7. Mention obligatoire

Ces scénarios sont des exemples pédagogiques. Ils doivent être adaptés au territoire, aux contraintes juridiques, aux moyens disponibles et aux décisions des parties concernées.
