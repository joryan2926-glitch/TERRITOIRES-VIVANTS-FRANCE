## 4. Pourquoi agir maintenant

Décrire les conséquences si aucune action n'est engagée :

- maintien de la vacance ;
- dégradation du patrimoine ;
- perte d'attractivité ;
- coûts futurs plus élevés ;
- ressources détruites ou dispersées ;
- difficulté à mobiliser les habitants ;
- opportunités de financement manquées.

## 7. Bénéfices attendus

### Pour la collectivité

- meilleure connaissance des ressources inutilisées ;
- coordination plus simple entre acteurs ;
- expérimentation cadrée ;
- appui à la revitalisation ;
- indicateurs de suivi.

### Pour les propriétaires

- interlocuteur identifié ;
- cadre de coopération ;
- valorisation d'un bien inutilisé ;
- visibilité sur les étapes.

### Pour les entreprises

- contribution RSE concrète ;
- valorisation de matériaux ou compétences ;
- participation à l'économie circulaire locale ;
- reporting plus clair.

### Pour les habitants

- lieux remis en usage ;
- services de proximité ;
- lien social ;
- amélioration du cadre de vie.

## 8. Décision attendue

- Accord pour une réunion de cadrage.
- Désignation d'un référent.
- Transmission de données ou documents disponibles.
- Identification d'un premier périmètre.
- Autorisation d'une visite ou d'un audit terrain.
