## 1. Objet du dossier

Ce dossier sert à préparer un premier échange avec Territoires Vivants France (TVF). Il permet à une collectivité, un propriétaire, une entreprise, une association, un citoyen, un bénévole, un artisan, un mécène ou un financeur de comprendre :

- qui contacter ;
- quelles informations transmettre ;
- quel service TVF peut mobiliser ;
- quel pôle ou quelle action correspond au besoin ;
- ce que l'interlocuteur peut apporter à TVF ;
- quels documents préparer avant une réunion ;
- quelle suite peut être donnée à la demande.

Ce document ne crée aucun engagement automatique. Il sert à qualifier une situation, orienter la demande et préparer une suite écrite si le dossier entre dans l'objet de TVF.

## 3. Résumé de la mission TVF

TVF a pour mission de remettre en usage les bâtiments, logements, commerces, friches, terrains et matériaux inutilisés au service des habitants et des territoires.

TVF agit comme plateforme de coopération territoriale. L'association ne remplace pas les collectivités, les services de l'État, les professionnels du bâtiment ou les dispositifs publics existants. Elle cherche à créer un cadre simple pour relier les acteurs concernés autour d'un dossier clair :

- observer ;
- qualifier ;
- prioriser ;
- mobiliser ;
- conventionner ;
- agir ;
- suivre l'impact.

## 5. Premier message à envoyer

Pour faciliter l'orientation, le premier message doit idéalement contenir :

- nom et prénom ;
- structure représentée ;
- fonction ;
- téléphone ;
- e-mail ;
- commune et territoire concerné ;
- type de demande ;
- description courte de la situation ;
- urgence éventuelle ;
- pièces disponibles ;
- objectif du contact ;
- disponibilité pour un échange.

### Modèle de message

Bonjour,

Je souhaite contacter Territoires Vivants France au sujet de : [logement vacant / commerce fermé / matériaux / friche / partenariat / financement / bénévolat / autre].

Territoire concerné : [commune, quartier, EPCI].  
Profil : [collectivité / propriétaire / entreprise / association / citoyen / financeur].  
Description : [présenter brièvement la situation].  
Pièces disponibles : [photos, adresse, documents, données, plans, inventaire].  
Objectif : [demander un échange, proposer un bien, proposer des matériaux, devenir partenaire, soutenir un projet].

Coordonnées : [nom, téléphone, e-mail].

Bien cordialement.

## 9. Ce que chaque public peut faire pour TVF

### Collectivités

Une collectivité peut :

- proposer un périmètre pilote ;
- partager des données publiques ou territoriales ;
- identifier des bâtiments, commerces, terrains ou friches à qualifier ;
- organiser une réunion de cadrage ;
- faciliter la mise en relation avec les acteurs locaux ;
- envisager une convention de coopération territoriale ;
- intégrer TVF dans une démarche de revitalisation, de réemploi ou de participation citoyenne.

### Propriétaires

Un propriétaire peut :

- proposer un logement vacant ;
- proposer un commerce fermé ;
- proposer un immeuble ou bâtiment inutilisé ;
- proposer un terrain ou une friche privée ;
- autoriser une première étude ;
- transmettre des photos, plans ou informations utiles ;
- envisager un usage temporaire ou solidaire dans un cadre écrit ;
- participer au programme Bien Solidaire à Usage Partagé.

### Entreprises

Une entreprise peut :

- proposer des matériaux de réemploi ;
- donner ou mettre à disposition du mobilier ;
- proposer des locaux vacants ;
- apporter des compétences ;
- participer au mécénat ;
- soutenir un projet local ;
- organiser une action RSE ;
- contribuer à la logistique ou au stockage.

### Artisans

Un artisan peut :

- réaliser une visite technique ;
- proposer un devis ;
- accompagner un chantier encadré ;
- former ou transmettre un savoir-faire ;
- conseiller sur le réemploi ;
- aider à évaluer la faisabilité d'un projet ;
- participer à une action locale.

### Associations

Une association peut :

- exprimer un besoin de local ;
- proposer un usage social, culturel, pédagogique ou solidaire ;
- mobiliser des bénévoles ;
- participer à l'animation d'un lieu ;
- accompagner des publics ;
- documenter les besoins du terrain.

### Bénévoles et citoyens

Un bénévole ou citoyen peut :

- signaler un lieu vacant ;
- signaler des matériaux disponibles ;
- participer à une mission encadrée ;
- aider à l'organisation d'événements ;
- participer à la communication ;
- contribuer à l'observation du territoire ;
- rejoindre une antenne locale lorsqu'elle sera structurée.

### Financeurs, mécènes et investisseurs solidaires

Un financeur peut :

- soutenir une phase de diagnostic ;
- financer une étude ;
- contribuer à un projet de remise en usage ;
- soutenir la matériauthèque ;
- accompagner la structuration nationale ;
- demander un reporting ;
- aider à financer des outils, documents, cartes ou indicateurs.

## 13. Règles de prudence

Avant toute publication ou annonce, TVF doit vérifier :

- l'accord du propriétaire ;
- l'autorisation d'utiliser les photos ;
- le statut réel du partenariat ;
- la faisabilité technique ;
- le cadre juridique ;
- les responsabilités ;
- les assurances ;
- les données personnelles ;
- la sécurité du site ;
- la distinction entre objectif, projet en instruction et résultat réalisé.

TVF ne doit pas annoncer de chiffres d'impact, partenaires, financements ou projets réalisés sans preuve vérifiable.

## 14. Synthèse opérationnelle

Pour contacter TVF efficacement :

1. Choisir le bon objet d'e-mail.
2. Présenter son profil.
3. Décrire le territoire ou le bien concerné.
4. Joindre les informations minimales.
5. Indiquer ce que l'on souhaite proposer.
6. Préciser la suite attendue : échange, étude, convention, partenariat, bénévolat, financement.
7. Attendre l'orientation TVF avant toute annonce publique.
