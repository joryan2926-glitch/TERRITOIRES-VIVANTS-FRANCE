# Registre des éléments officiels en attente

Ce registre liste les éléments réels à compléter uniquement lorsqu'ils seront disponibles, vérifiés et autorisés à être publiés. Il évite d'inventer des informations administratives, partenaires, financeurs ou résultats.

---

## 1. Règle générale

Un élément ne doit être publié sur le site TVF que si :

- la source est vérifiée ;
- la date est connue ;
- l'autorisation de publication existe si nécessaire ;
- le document justificatif est archivé ;
- la formulation ne crée pas de confusion.

---

## 2. Éléments administratifs à compléter

| Élément | Statut | Preuve attendue | Page à mettre à jour |
|---|---|---|---|
| Numéro RNA | En attente de disponibilité officielle | Récépissé ou publication officielle | Mentions légales, transparence |
| Numéro SIREN | En attente si attribution | Avis INSEE ou document officiel | Mentions légales |
| Statut exact de l'association | En attente de formalisation ou validation définitive | Statuts adoptés / récépissé | Qui sommes-nous, mentions légales |
| Hébergeur | À renseigner avec les informations légales exactes | Mentions fournisseur / contrat / tableau de bord | Mentions légales |
| Responsables officiels actualisés | À confirmer après décision interne | PV, statuts ou décision | Gouvernance, mentions légales |

---

## 3. Partenaires et financeurs

| Élément | Peut être publié ? | Condition |
|---|---|---|
| Logo partenaire | Non sans accord écrit | Convention, courrier ou autorisation |
| Nom d'une collectivité | Non sans validation | Accord formel ou communication conjointe |
| Financeur pressenti | Non | Attendre décision ou convention |
| Mécène | Non sans accord | Convention de mécénat ou autorisation |
| Entreprise contributrice | Non sans accord | Fiche contribution et validation communication |

---

## 4. Projets et résultats

| Type d'information | Statut publiable | Preuve minimale |
|---|---|---|
| Signalement | Non public par défaut | Fiche signalement interne |
| Projet en instruction | Public seulement si accord | Fiche projet + validation |
| Projet conventionné | Public possible selon convention | Convention signée |
| Action réalisée | Public possible après vérification | Compte rendu + preuves |
| Indicateur d'impact | Public après calcul documenté | Grille d'impact + source |

---

## 5. Contrôle avant mise à jour publique

Avant de publier un nouvel élément :

- [ ] source vérifiée ;
- [ ] date vérifiée ;
- [ ] autorisation écrite si nécessaire ;
- [ ] formulation prudente ;
- [ ] page concernée identifiée ;
- [ ] document archivé ;
- [ ] validation interne notée dans le registre des décisions.

---

## 6. Comment utiliser ce registre

Ce registre doit être relu avant toute mise à jour des pages :

- Qui sommes-nous ;
- Gouvernance ;
- Transparence ;
- Mentions légales ;
- Partenaires ;
- Impact ;
- Kit média ;
- Dossier de présentation.
