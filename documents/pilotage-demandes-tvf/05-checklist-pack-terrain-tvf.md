# TERRITOIRES VIVANTS FRANCE
## Checklist pack terrain TVF

Ce document liste les pieces a emporter ou preparer avant une rencontre, une visite, une collecte ou une action terrain.

## Avant rendez-vous institutionnel

- [ ] Dossier de presentation TVF
- [ ] Synthese collectivite 1 page
- [ ] Fiche territoire partenaire
- [ ] Note d'opportunite territoriale
- [ ] Carte ou perimetre si disponible
- [ ] Questions a poser
- [ ] Compte rendu pret a remplir

## Avant visite d'un bien

- [ ] Formulaire F-03 proprietaire
- [ ] Autorisation de visite
- [ ] Fiche audit terrain
- [ ] Evaluation securite site
- [ ] Appareil photo ou telephone charge
- [ ] Coordonnees du proprietaire ou mandataire
- [ ] Equipements de securite si necessaires

## Avant collecte de materiaux

- [ ] Formulaire F-04 materiaux
- [ ] Criteres d'acceptation materiaux
- [ ] Bordereau de don ou remise
- [ ] Photos avant retrait
- [ ] Conditions de transport
- [ ] Lieu de stockage confirme
- [ ] Personne responsable du retrait

## Avant action benevole ou chantier

- [ ] Fiche mission benevole
- [ ] Feuille d'emargement
- [ ] Consignes securite
- [ ] Plan de prevention si necessaire
- [ ] Autorisation droit image
- [ ] Responsable terrain identifie
- [ ] Trousse de premiers secours si action physique

## Apres action

- [ ] Compte rendu action terrain
- [ ] Photos classees
- [ ] Decisions notees
- [ ] Pieces manquantes listees
- [ ] Relance planifiee
- [ ] Statut mis a jour dans le registre
