# TERRITOIRES VIVANTS FRANCE
## Matrice formulaire vers suite TVF

Ce document indique quelle suite donner a chaque formulaire recu.

| Code | Formulaire | Suite immediate | Pieces prioritaires | Document suivant | Decision possible |
|---|---|---|---|---|---|
| F-01 | Contact general | Qualifier et orienter | Message, profil, contact | Formulaire specialise si besoin | Orientation / classement |
| F-02 | Collectivite territoire partenaire | Proposer rendez-vous de cadrage | Note besoin, territoire, service referent | Fiche territoire partenaire | Etude / convention cooperation |
| F-03 | Proprietaire bien vacant | Demander pieces et photos | Propriete, adresse, photos | Fiche bien solidaire | Visite / refus / etude |
| F-04 | Proposition materiaux | Qualifier acceptabilite | Photos, quantite, etat, retrait | Bordereau don materiaux | Acceptation / refus / attente projet |
| F-05 | Entreprise partenariat | Qualifier contribution | Type apport, volume, calendrier | Fiche entreprise | Convention partenariat |
| F-06 | Local stockage | Verifier securite et acces | Photos, surface, duree, mandat | Protocole local stockage | Visite / convention mise a disposition |
| F-07 | Vehicule logistique | Verifier assurance et usage | Fiche vehicule, conditions | Accord logistique | Accord ponctuel / refus |
| F-08 | Benevole | Orienter mission | Disponibilites, competences | Fiche mission benevole | Mission / attente / refus |
| F-09 | Association insertion chantier | Etudier cadre d'action | Assurance, encadrement, public | Cadre chantier insertion | Convention / plan prevention |
| F-10 | Financeur mecene | Preparer note d'opportunite | Criteres, calendrier, montant | Dossier financement | Rendez-vous / depot dossier |
| F-11 | Signalement citoyen | Verifier sans publier | Localisation, photos publiques | Protocole qualification signalement | Qualification / archive |
| F-12 | Presse institutionnel | Repondre avec elements officiels | Sujet, delai, media | Kit media | Interview / reponse ecrite |

## Delais de traitement recommandes

- P1 : premiere reponse sous 48 heures ouvrees.
- P2 : premiere reponse sous 5 jours ouvres.
- P3 : premiere reponse sous 7 a 10 jours ouvres.

## Regle de decision

La suite d'un formulaire doit toujours etre classee dans une des categories suivantes :

- a completer ;
- rendez-vous ;
- instruction interne ;
- reorientation ;
- refus motive ;
- archivage ;
- passage en convention.
