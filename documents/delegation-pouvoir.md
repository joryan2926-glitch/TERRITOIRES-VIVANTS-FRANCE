- dépôt d'un dossier ;
- demande de subvention ;
- retrait ou dépôt de pièces ;
- échange avec une collectivité ;
- suivi d'un projet ;
- autre :

## 5. Limites de la délégation

Le délégataire ne peut pas :

- engager une dépense non autorisée ;
- signer une convention hors périmètre ;
- annoncer un partenariat non validé ;
- utiliser le logo TVF en dehors du cadre prévu ;
- modifier les statuts ou règles internes ;
- prendre une décision relevant d'une instance compétente.
