# Protocole d'officialisation d'un partenariat - TVF

Document préparatoire modifiable - version de travail

## 1. Objet

Ce protocole fixe les étapes à respecter avant d'afficher publiquement un partenaire, d'utiliser un logo, de publier une citation, d'annoncer une contribution ou de communiquer sur une coopération.

Il protège TVF, le partenaire et les publics concernés contre les annonces prématurées ou imprécises.

## 2. Principe général

Un acteur n'est considéré comme partenaire public de TVF que lorsque les éléments suivants sont réunis :

- accord écrit ou convention signé ;
- objet de la coopération clairement défini ;
- responsabilités identifiées ;
- modalités de communication validées ;
- usage du logo autorisé ;
- absence de chiffre, résultat ou contribution inventée ;
- validation interne TVF.

## 4. Points à vérifier avant publication

- [ ] Le partenariat est réel et formalisé.
- [ ] Le nom exact du partenaire est validé.
- [ ] Le logo transmis est officiel.
- [ ] Le droit d'utiliser le logo est confirmé.
- [ ] La formulation publique est relue par le partenaire.
- [ ] Les dates et périmètres sont exacts.
- [ ] Aucun montant non confirmé n'est annoncé.
- [ ] Aucun résultat non mesuré n'est annoncé.
- [ ] Les photos éventuelles sont autorisées.
- [ ] La durée ou limite du partenariat est claire.

## 5. Formulations recommandées

Formulations prudentes :

- "TVF et [nom] ont signé une convention de coopération portant sur [objet]."
- "[Nom] contribue à [action précise] dans le cadre d'un accord formalisé."
- "Cette coopération vise à [objectif], selon des modalités définies par écrit."

Formulations à éviter sans preuve :

- "Partenaire officiel" sans accord signé.
- "Soutien majeur" sans cadre précis.
- "Financeur" sans contribution reçue ou conventionnée.
- "Projet réalisé avec" lorsque le projet est seulement en instruction.
- "Impact obtenu" sans indicateur vérifié.
