# Convention de coopération territoriale TVF

Document préparatoire modifiable - version de travail

## 1. Objet

Cette convention préparatoire sert à cadrer une coopération entre Territoires Vivants France et une collectivité ou un organisme territorial. Elle doit être adaptée au contexte, aux compétences de chaque partie et aux règles applicables.

## 3. Contexte territorial

Décrire le contexte :

- besoins du territoire ;
- enjeux de vacance immobilière ;
- commerces ou locaux inoccupés ;
- friches ou terrains inutilisés ;
- matériaux et économie circulaire ;
- insertion ou mobilisation citoyenne ;
- programmes publics existants.

## 4. Objectifs de coopération

Les parties souhaitent coopérer pour :

- qualifier les besoins territoriaux ;
- repérer les ressources inutilisées ;
- organiser des réunions de cadrage ;
- préparer des fiches projet ;
- mobiliser les acteurs locaux ;
- étudier les conditions de convention ;
- suivre les indicateurs.

## 5. Rôle de TVF

TVF peut contribuer à :

- structurer la méthode ;
- préparer les documents ;
- analyser les signalements ;
- aider à la coordination ;
- proposer des indicateurs ;
- accompagner une expérimentation ;
- produire une note de restitution.

TVF ne se substitue pas aux décisions de la collectivité, aux procédures publiques, aux services compétents ou aux professionnels réglementés.

## 6. Rôle de la collectivité

La collectivité peut contribuer à :

- désigner un référent ;
- transmettre des informations disponibles ;
- identifier des secteurs prioritaires ;
- faciliter la mise en relation avec des acteurs locaux ;
- participer aux réunions de suivi ;
- valider les communications publiques qui la concernent ;
- étudier les suites possibles.

## 8. Données et confidentialité

Les données transmises doivent être utilisées uniquement dans le cadre de la coopération. Les informations sensibles, adresses privées, noms de propriétaires ou données personnelles ne peuvent pas être publiés sans accord.

## 9. Communication

Toute communication publique mentionnant l'autre partie doit être validée avant diffusion. Aucun logo ne doit être utilisé sans autorisation.
