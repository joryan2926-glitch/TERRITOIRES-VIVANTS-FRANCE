# Pièces à fournir selon le type de demande

Document préparatoire modifiable - version de travail

## 1. Objectif

Cette checklist permet de préparer un dossier complet avant instruction par Territoires Vivants France. Elle évite les échanges incomplets et permet de qualifier rapidement la demande.

## 2. Propriétaire proposant un bien

| Pièce | Obligatoire | Commentaire |
| --- | --- | --- |
| Identité du propriétaire ou représentant | Oui | Nom, prénom, structure, coordonnées |
| Adresse complète du bien | Oui | Numéro, rue, commune, code postal |
| Justificatif de propriété ou mandat | À vérifier | Titre, taxe foncière, mandat, attestation |
| Description du bien | Oui | Logement, commerce, bâtiment, terrain, friche |
| Photos récentes | Oui | Façade, accès, intérieur si possible |
| État apparent | Oui | Bon état, dégradé, inaccessible, dangereux |
| Contraintes connues | Oui | Servitudes, copropriété, sécurité, occupation |
| Usage souhaité ou accepté | Recommandé | Logement, commerce, association, tiers-lieu |
| Durée envisageable de coopération | Recommandé | À discuter par convention |
| Documents techniques existants | Si disponibles | Diagnostics, plans, devis, études |

## 3. Collectivité

| Pièce | Obligatoire | Commentaire |
| --- | --- | --- |
| Contact référent | Oui | Élu, direction, service technique ou projet |
| Périmètre territorial | Oui | Commune, quartier, rue, secteur prioritaire |
| Besoin public identifié | Oui | Habitat, commerce, friche, matériaux, insertion |
| Données disponibles | Si disponibles | Vacance, friches, commerces, habitat, foncier |
| Programmes existants | Recommandé | PLH, OPAH, Action Cœur de Ville, NPNRU, PCAET |
| Biens ou ressources ciblés | Recommandé | Liste, carte, photos, contexte |
| Contraintes administratives | À préciser | Marchés, domanialité, sécurité, urbanisme |
| Attentes envers TVF | Oui | Diagnostic, animation, coordination, documents |
| Calendrier souhaité | Recommandé | Priorités, échéances, réunions |

## 4. Entreprise proposant des matériaux ou compétences

| Pièce | Obligatoire | Commentaire |
| --- | --- | --- |
| Identification de l'entreprise | Oui | Nom, SIRET si disponible, contact |
| Type de contribution | Oui | Matériaux, mobilier, locaux, mécénat, compétences |
| Description détaillée | Oui | Nature, quantité, état, localisation |
| Photos | Recommandé | Vue générale et détails |
| Disponibilité | Oui | Date, durée, urgence d'enlèvement |
| Conditions logistiques | Oui | Accès, manutention, transport, horaires |
| Restrictions d'usage | À préciser | Sécurité, conformité, responsabilité |
| Attentes de valorisation | Recommandé | RSE, reporting, communication validée |

## 5. Association, citoyen ou bénévole

| Pièce | Obligatoire | Commentaire |
| --- | --- | --- |
| Coordonnées | Oui | Nom, e-mail, téléphone si possible |
| Type de contribution | Oui | Signalement, bénévolat, usage, relais local |
| Description de la situation | Oui | Besoin, lieu, public concerné |
| Photos ou documents | Si disponibles | Uniquement si autorisés |
| Disponibilités | Recommandé | Réunion, visite, action encadrée |
| Compétences utiles | Recommandé | Technique, juridique, social, communication |

## 6. Règles de prudence

- Ne pas entrer dans un bien privé sans autorisation.
- Ne pas transmettre de données personnelles inutiles.
- Ne pas publier de photo identifiable sans accord.
- Ne pas présenter une intention comme un engagement.
- Ne pas engager de dépense avant validation du cadre.
