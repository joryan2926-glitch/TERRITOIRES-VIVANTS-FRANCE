## 5. Points à vérifier

- [ ] Les devis portent bien sur le même périmètre.
- [ ] Les options sont comparables.
- [ ] Les frais annexes sont inclus ou identifiés.
- [ ] Les délais sont compatibles avec le projet.
- [ ] Les assurances ou qualifications sont précisées si nécessaire.
- [ ] Les réserves du prestataire sont comprises.
- [ ] Les conditions de paiement sont acceptables.
- [ ] La dépense est compatible avec le budget disponible.
- [ ] Aucun conflit d'intérêt n'est identifié.
