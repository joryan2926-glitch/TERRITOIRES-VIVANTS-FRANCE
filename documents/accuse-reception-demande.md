## 3. Modèle de message

Madame, Monsieur,

Territoires Vivants France accuse réception de votre demande transmise le `[date]`, concernant `[objet de la demande]` sur le territoire de `[commune ou secteur]`.

Votre demande est enregistrée sous la référence :

`[numéro de dossier]`

Elle fera l'objet d'une première analyse afin de vérifier :

- la nature exacte du besoin ;
- les informations déjà disponibles ;
- les pièces utiles à l'instruction ;
- les contraintes éventuelles ;
- l'adéquation avec les missions de Territoires Vivants France ;
- les suites possibles à proposer.

## 4. Rappel de prudence

L'enregistrement de cette demande ne constitue pas une validation de projet, un engagement financier, une promesse d'intervention, une convention ou une acceptation automatique.

Toute suite opérationnelle devra être précédée, selon le cas, d'une qualification du dossier, d'une visite, d'une analyse des risques, d'une décision d'orientation et d'un cadre écrit adapté.
