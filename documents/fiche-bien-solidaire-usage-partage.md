# Fiche Bien Solidaire à Usage Partagé

Document préparatoire modifiable - version de travail

## 1. Objectif

Cette fiche permet d'étudier un bien proposé par un propriétaire dans le cadre d'un usage temporaire, solidaire ou partagé. Elle ne crée aucun engagement automatique : le propriétaire conserve la propriété du bien et toute utilisation doit être formalisée par convention.

## 7. Cadre de coopération envisagé

Le cadre reste à définir par convention. Les options possibles peuvent inclure :

- rénovation contre usage temporaire ;
- rénovation contre loyer solidaire ;
- rénovation contre partage de revenus ;
- mise à disposition partielle ;
- occupation transitoire ;
- convention de valorisation territoriale.

TVF ne doit jamais utiliser le terme "contrepartie" comme principe automatique. Les modalités doivent être décrites comme une coopération, un engagement réciproque ou une valorisation partagée.
