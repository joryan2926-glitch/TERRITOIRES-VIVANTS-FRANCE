# Cadre de cooperation - Chantiers d'insertion et actions terrain

Document operationnel modifiable

---

## 1. Principe

TVF peut contribuer a identifier des besoins territoriaux et a preparer des actions utiles, mais ne doit pas improviser un chantier d'insertion sans structure competente, encadrement, assurance et cadre ecrit.

Les structures d'insertion, associations habilitees, dispositifs emploi et acteurs sociaux doivent etre associes selon leurs competences et leurs autorisations.

---

## 2. Actions compatibles a etudier

| Action | Conditions minimales |
|---|---|
| Tri de materiaux | Local securise, referent, equipements, registre |
| Inventaire de ressources | Photos autorisees, fiche materiaux, consignes |
| Entretien leger d'espace | Autorisation du site, risques verifies |
| Petits travaux non dangereux | Encadrement qualifie, assurance, consignes |
| Atelier pedagogique | Objectif clair, public encadre, lieu adapte |
| Logistique legere | Charge, transport, securite et responsabilites definies |

---

## 3. Actions a exclure sans expertise

| Action | Motif |
|---|---|
| Travaux lourds | Responsabilite, assurance, competence technique |
| Electricite, gaz, structure | Risque majeur |
| Depose dangereuse | Reglementation et securite |
| Site instable ou pollue | Evaluation technique obligatoire |
| Intervention sans autorisation | Risque juridique |
| Encadrement de public fragile sans partenaire habilite | Responsabilite sociale et securite |

---

## 4. Questions avant tout chantier

| Question | Reponse |
|---|---|
| Qui est maitre du site ? | |
| Qui autorise l'action ? | |
| Qui encadre les personnes ? | |
| Quelle assurance couvre l'action ? | |
| Quels risques sont identifies ? | |
| Quels equipements sont necessaires ? | |
| Quelle duree est prevue ? | |
| Quel compte rendu sera produit ? | |

---

## 5. Documents obligatoires selon le cas

- Fiche projet.
- Evaluation securite site.
- Plan de prevention action terrain.
- Consignes securite action terrain.
- Feuille d'emargement.
- Fiche mission benevole.
- Compte rendu action terrain.
- Fiche incident action si necessaire.

---

## 6. Decision TVF possible

| Decision | Signification |
|---|---|
| Action possible | Conditions minimales reunies |
| Action a completer | Des pieces ou validations manquent |
| Action a confier | Une structure specialisee doit porter l'action |
| Action refusee | Risque trop important ou cadre insuffisant |
