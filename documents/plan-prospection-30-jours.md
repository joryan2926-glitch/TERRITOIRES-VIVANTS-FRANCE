# Plan de prospection 30 jours - TVF

Document opérationnel modifiable - version de travail

Objectif : organiser les premiers contacts de Territoires Vivants France de manière structurée, crédible et traçable.

Important : ce plan ne fixe aucun objectif chiffré obligatoire. Les volumes de contacts, rendez-vous ou relances doivent être adaptés aux capacités réelles de TVF.

## 1. Principe général

La prospection TVF ne doit pas ressembler à une campagne commerciale classique.

Elle doit être :

- ciblée ;
- documentée ;
- respectueuse ;
- institutionnelle ;
- progressive ;
- suivie dans un registre ;
- cohérente avec la charte éthique.

L'objectif n'est pas d'obtenir un engagement immédiat, mais d'ouvrir des échanges sérieux avec les bons interlocuteurs.

## 2. Publics prioritaires

| Public | Objectif du premier contact | Support recommandé |
| --- | --- | --- |
| Commune ou EPCI | Comprendre les besoins du territoire | Courrier collectivité + fiche territoire |
| Propriétaire | Étudier un bien vacant ou inutilisé | Courrier propriétaire + fiche propriétaire |
| Entreprise | Identifier matériaux, compétences ou mécénat | Courrier entreprise + fiche contribution |
| Association | Identifier besoins d'usage ou relais local | Fiche partenaire potentiel |
| Financeur ou mécène | Présenter la méthode et les conditions de soutien | Courrier financeur + note d'impact |
| Citoyen ou bénévole | Recueillir une disponibilité ou un signalement | Fiche bénévole ou signalement |

## 3. Semaine 1 - Préparer

### Objectif

Disposer d'une base de contacts propre et de supports prêts à envoyer.

### Actions

- relire le pitch officiel ;
- sélectionner les publics prioritaires ;
- préparer une liste de contacts ;
- vérifier les adresses e-mail et fonctions ;
- préparer les documents adaptés ;
- créer un registre de suivi ;
- définir le message principal par cible.

### Documents utiles

- `pitch-officiel-tvf.md`
- `argumentaires-par-public.md`
- `scripts-prise-contact.md`
- `registre-demandes-entrantes.md`
- `fiche-partenaire-potentiel.md`

### Points de vigilance

- ne pas envoyer de message générique ;
- ne pas annoncer de partenariat ;
- ne pas promettre de financement ;
- ne pas afficher de faux résultats.

## 4. Semaine 2 - Contacter

### Objectif

Envoyer les premiers messages personnalisés et tracer les réponses.

### Actions

- envoyer les e-mails ou messages LinkedIn ;
- adapter le message à chaque destinataire ;
- enregistrer la date d'envoi ;
- noter le canal utilisé ;
- classer le contact par profil ;
- préparer les relances.

### Champs à suivre

- date d'envoi ;
- destinataire ;
- structure ;
- profil ;
- sujet ;
- canal ;
- statut ;
- prochaine action ;
- responsable TVF.

## 5. Semaine 3 - Relancer

### Objectif

Relancer sans pression et proposer un échange court.

### Actions

- relancer les contacts sans réponse ;
- répondre aux premiers retours ;
- proposer un créneau de rendez-vous ;
- envoyer une fiche de cadrage si nécessaire ;
- préparer les questions de rendez-vous.

### Documents utiles

- `scripts-prise-contact.md`
- `fiche-preparation-rendez-vous.md`
- `scripts-rendez-vous-institutionnels.md`
- `objections-reponses.md`

## 6. Semaine 4 - Qualifier

### Objectif

Transformer les réponses en dossiers lisibles.

### Actions

- organiser les rendez-vous ;
- produire un compte rendu ;
- demander les pièces utiles ;
- qualifier les besoins ;
- classer les suites : à compléter, à instruire, à réorienter, à mettre en attente ou sans suite ;
- mettre à jour le registre.

### Documents utiles

- `fiche-projet.md`
- `fiche-collectivite.md`
- `fiche-proprietaire.md`
- `fiche-entreprise.md`
- `fiche-decision-orientation.md`
- `registre-suivi-decisions.md`

## 8. Règles de relance

- rester poli et bref ;
- ne pas relancer trop souvent ;
- rappeler l'objet en une phrase ;
- proposer une sortie simple : échange, orientation ou absence de suite ;
- tracer chaque relance.

## 9. Message de relance court

Bonjour,

Je me permets de revenir vers vous concernant mon message sur Territoires Vivants France.

L'objectif est simplement de vérifier si notre démarche peut être utile à votre territoire ou à votre structure : biens vacants, matériaux réemployables, coopération locale, mécénat ou engagement citoyen.

Je reste disponible pour un échange court, sans engagement.

Cordialement,
