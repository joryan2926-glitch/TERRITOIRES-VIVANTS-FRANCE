# Demande de devis ou prestation - TVF

Document préparatoire modifiable - version de travail

## 1. Objet

Ce modèle permet de demander un devis, une proposition technique ou une estimation à une entreprise, un artisan, un bureau d'études, un prestataire ou un fournisseur dans le cadre d'un projet Territoires Vivants France.

Il ne vaut pas commande. Aucun engagement de dépense ne doit être pris avant validation interne, vérification du budget disponible et cadre écrit adapté.

## 7. Critères de comparaison

Le choix ne doit pas reposer uniquement sur le prix. Les critères peuvent inclure :

- compréhension du besoin ;
- capacité à intervenir dans le délai ;
- clarté du devis ;
- expérience sur des sites similaires ;
- sécurité ;
- qualité des matériaux ou méthodes ;
- impact environnemental ;
- capacité de reporting ;
- garanties ;
- coût total.

## 8. Message type

Madame, Monsieur,

Territoires Vivants France étudie actuellement le dossier `[référence]`, concernant `[description courte du projet]` sur le territoire de `[commune]`.

Dans ce cadre, nous souhaitons obtenir une proposition ou un devis portant sur :

`[description précise de la prestation]`

Cette demande ne vaut pas commande. Elle vise à disposer d'une estimation claire afin d'instruire le dossier, construire un budget prévisionnel et vérifier les conditions de faisabilité.

Merci d'indiquer dans votre retour :

- le détail de la prestation ;
- le prix HT et TTC ;
- les délais ;
- les hypothèses retenues ;
- les limites éventuelles ;
- les conditions de validité du devis ;
- les assurances ou qualifications utiles.

Bien cordialement,

`[Nom et fonction]`  
Territoires Vivants France
