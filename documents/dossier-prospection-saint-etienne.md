## 1. Objectif du dossier

Ce dossier sert a organiser les premiers contacts necessaires au lancement operationnel de Territoires Vivants France (TVF) a Saint-Etienne.

Il ne presente aucun acteur comme partenaire officiel. Il recense des interlocuteurs publics, economiques, associatifs et techniques a solliciter pour verifier l'interet d'une cooperation, qualifier les besoins du territoire et identifier les premieres situations pouvant entrer dans l'objet de TVF.

Le dossier doit permettre de repondre a quatre questions simples :

- qui contacter en premier ;
- pourquoi cet acteur est utile au lancement de TVF ;
- ce que TVF peut lui proposer concretement ;
- quelle suite demander apres le premier echange.

---

## 2. Regle de prudence

Les structures citees dans ce document sont des contacts a solliciter, pas des partenaires declares.

Avant toute communication publique, TVF doit obtenir :

- un accord ecrit ;
- une validation du perimetre de cooperation ;
- une convention ou un courrier d'intention lorsque cela est necessaire ;
- une autorisation d'utiliser un nom, un logo, une photographie ou une reference.

Formulation recommandee : "TVF souhaite rencontrer les acteurs du territoire afin d'etudier les cooperations possibles."

Formulation a eviter : "TVF travaille avec..." tant qu'aucun accord n'est signe.

## 8. Arguments par public

### Collectivites

Ce que TVF peut apporter :

- une methode de reperage et de qualification des lieux inutilises ;
- un relais citoyen pour faire remonter des situations ;
- une coordination entre proprietaires, artisans, associations et financeurs ;
- des documents de cadrage pour eviter les projets flous ;
- des indicateurs d'impact lisibles.

Ce qu'il faut demander :

- un rendez-vous de cadrage ;
- les priorites publiques du territoire ;
- les dispositifs deja existants ;
- les contacts techniques pertinents ;
- les limites a respecter.

Message cle :

"TVF ne remplace pas la collectivite. TVF peut aider a transformer des situations dispersees en dossiers lisibles, encadres et orientables."

### Entreprises

Ce que TVF peut apporter :

- une valorisation territoriale des materiaux, mobiliers ou locaux inutilises ;
- un cadre de contribution clair ;
- une orientation vers des projets utiles ;
- une trace documentaire de la contribution ;
- une logique RSE concrete, locale et verifiable.

Ce qu'il faut demander :

- types de ressources disponibles ;
- volumes approximatifs ;
- conditions de retrait ;
- contraintes juridiques ou logistiques ;
- accord ou non pour mention publique ulterieure.

Message cle :

"Une ressource inutilisee peut devenir utile si elle est qualifiee, tracee et affectee a un projet valide."

### Proprietaires

Ce que TVF peut apporter :

- une premiere qualification du bien ;
- un scenario d'usage possible ;
- une mise en relation avec les bons acteurs ;
- un cadre de convention si le dossier est recevable ;
- une solution progressive sans engagement automatique.

Ce qu'il faut demander :

- adresse du bien ;
- statut de propriete ;
- etat connu ;
- disponibilite ;
- contraintes ;
- photos autorisees ;
- accord de principe pour visite si pertinent.

Message cle :

"Le proprietaire conserve son bien. Toute suite passe par une analyse, une convention et une decision explicite."

### Associations

Ce que TVF peut apporter :

- identification de locaux ou espaces utiles ;
- projets de quartier ;
- chantiers participatifs ;
- mise en reseau ;
- materiaux pour projets valides.

Ce qu'il faut demander :

- besoin d'espace ;
- publics accompagnes ;
- capacite d'animation ;
- assurances ;
- encadrement ;
- contraintes d'accueil.

Ces rendez-vous doivent permettre d'identifier un premier portefeuille de situations :

- un logement ou immeuble vacant volontaire ;
- un commerce vacant ou rez-de-chaussee a remettre en usage ;
- une ressource de materiaux reutilisables ;
- une friche ou un terrain a qualifier ;
- une structure associative ou d'insertion capable de participer.

Une fois ces cinq axes qualifies, TVF pourra produire une premiere note de lancement Saint-Etienne avec acteurs rencontres, besoins exprimes, opportunites, risques, documents a reunir et calendrier de travail.
