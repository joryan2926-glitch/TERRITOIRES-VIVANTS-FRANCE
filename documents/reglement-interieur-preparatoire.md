# Règlement intérieur préparatoire TVF

Document préparatoire modifiable - version de travail

## 1. Objet

Ce règlement intérieur préparatoire précise les règles de fonctionnement internes de Territoires Vivants France avant adoption d'une version officielle par les instances compétentes.

## 2. Principes de fonctionnement

- Les décisions doivent être prises dans l'intérêt de l'association et de son objet.
- Les projets sont instruits avant communication publique.
- Les engagements sont formalisés par écrit lorsque cela est nécessaire.
- Les bénévoles ne peuvent intervenir que dans un cadre validé.
- Les partenaires ne sont affichés publiquement qu'après accord réel.

## 3. Organisation des dossiers

Chaque dossier doit comprendre, lorsque c'est pertinent :

- une fiche de demande ;
- un numéro de dossier ;
- un référent identifié ;
- un état des lieux ;
- les pièces transmises ;
- les échanges importants ;
- une décision ou orientation ;
- les documents de convention ;
- les indicateurs de suivi.

## 4. Règles applicables aux bénévoles

Un bénévole ne doit pas :

- entrer dans un bien privé sans autorisation ;
- intervenir sur un chantier sans encadrement ;
- représenter TVF officiellement sans mandat ;
- utiliser le logo TVF sans accord ;
- publier des informations sensibles ou non vérifiées.

Un bénévole peut :

- aider à documenter un besoin ;
- relayer une information validée ;
- participer à une réunion ;
- contribuer à une action encadrée ;
- aider à la préparation de documents.

## 5. Communication publique

Toute communication doit respecter les règles suivantes :

- pas de faux partenaire ;
- pas de faux financeur ;
- pas de chiffre d'impact inventé ;
- pas de promesse non conventionnée ;
- distinction claire entre objectif, projet en instruction et action réalisée.

## 6. Gestion des conflits d'intérêt

Toute personne impliquée dans un dossier doit signaler une situation pouvant créer un conflit d'intérêt :

- lien familial ;
- intérêt financier ;
- relation professionnelle ;
- avantage personnel ;
- implication dans une structure concernée.

## 7. Documents et données

Les documents transmis à TVF doivent être utilisés uniquement pour l'instruction du dossier concerné. Les données sensibles ne doivent pas être publiées sans accord.

## 8. Mise à jour

Ce règlement doit être adapté après déclaration officielle, structuration des instances, adoption des statuts définitifs et évolution des activités.
