## 4. Engagements de TVF

TVF peut notamment s'engager à :

- qualifier la situation ;
- coordonner les échanges ;
- préparer un dossier de projet ;
- identifier des ressources ;
- suivre les étapes ;
- documenter les décisions ;
- rendre compte des actions réalisées.

## 5. Engagements du partenaire

Le partenaire peut notamment s'engager à :

- fournir des informations exactes ;
- désigner un interlocuteur ;
- faciliter les visites ou échanges ;
- respecter les règles convenues ;
- informer TVF de tout changement important ;
- ne pas communiquer publiquement avant validation commune.
