# Bibliotheque interne TVF OS

Documents internes prets a remplir pour recevoir, qualifier, instruire, conventionner et suivre les demandes TVF.

## Categories

- `00-index` : index du kit et index complet.
- `01` a `12` : formulaires par type de demande.
- `13-conventions-types` : conventions, autorisations, chartes et PV types.
- `14-listes-pieces` : pieces a fournir par sujet.
- `15-courriers-prets-a-envoyer` : courriers types.
- `16-instruction-dossier` : fiches internes d'instruction.

## Regle

Completer le numero de dossier TVF, les coordonnees, les pieces recues, la decision et les signatures si necessaire. Adapter tout document engageant avant transmission ou signature.
