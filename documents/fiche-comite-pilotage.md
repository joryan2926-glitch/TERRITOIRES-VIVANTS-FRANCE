## 1. Objectif

Cette fiche permet de structurer un comité de pilotage territorial autour d'un diagnostic, d'un projet pilote ou d'une coopération TVF. Elle clarifie les participants, rôles, décisions et livrables.

## 4. Rôle du comité

Le comité de pilotage peut :

- suivre l'avancement ;
- valider les étapes de diagnostic ;
- prioriser les pistes ;
- lever les blocages ;
- sécuriser la communication ;
- préparer les conventions ;
- suivre les indicateurs.

Le comité ne remplace pas les décisions formelles des autorités ou instances compétentes.

## 7. Règles de fonctionnement

- Les informations sensibles restent internes.
- Les décisions sont consignées par écrit.
- Les logos et noms des partenaires ne sont utilisés qu'après accord.
- Les données publiées doivent être sourcées.
- Les désaccords ou réserves sont notés.
