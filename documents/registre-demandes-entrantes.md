# Registre des demandes entrantes - TVF

Document de suivi administratif modifiable - version de travail

## 1. Objet du registre

Ce registre permet de tracer toutes les demandes reçues par Territoires Vivants France avant instruction : signalement, proposition de bien, contribution de matériaux, demande de collectivité, candidature bénévole, offre de mécénat ou demande de partenariat.

Il sert à éviter les pertes d'information, à identifier les priorités, à protéger les données personnelles et à distinguer clairement :

- une demande reçue ;
- une demande complète ;
- un dossier en instruction ;
- un projet orienté ;
- un projet validé ;
- un dossier classé sans suite.

## 2. Règle de numérotation

Format recommandé :

`TVF-AAAA-TERRITOIRE-TYPE-NNN`

Exemples :

- `TVF-2026-STE-HAB-001`
- `TVF-2026-STE-MAT-002`
- `TVF-2026-NA-COL-003`

Codes possibles :

| Code | Signification |
| --- | --- |
| HAB | Logement, immeuble, bien bâti |
| COM | Commerce ou local d'activité |
| FRI | Friche, terrain, bâtiment abandonné |
| MAT | Matériaux, mobilier, équipements |
| COL | Collectivité ou territoire |
| ENT | Entreprise ou mécénat |
| BEN | Bénévole ou citoyen |
| FIN | Financeur, fondation ou mécène |

## 7. Protection des données

Les informations personnelles doivent être limitées à ce qui est nécessaire à l'instruction. Les données ne doivent pas être diffusées publiquement sans accord explicite.

Points à contrôler :

- [ ] La personne sait pourquoi ses données sont collectées
- [ ] Les pièces sensibles sont conservées avec prudence
- [ ] Les photos ne sont pas publiées sans autorisation
- [ ] Les coordonnées ne sont pas partagées sans nécessité
- [ ] Le dossier peut être supprimé ou archivé sur demande légitime

## 8. Revue périodique

Fréquence recommandée :

- revue hebdomadaire des nouvelles demandes ;
- revue mensuelle des dossiers en attente ;
- archivage trimestriel des demandes sans suite ;
- consolidation annuelle des typologies de demandes.
