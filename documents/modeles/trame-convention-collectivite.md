# Trame - Convention de cooperation territoriale

Statut : modele de travail a adapter.  
Ce document aide a cadrer une cooperation entre une collectivite et Territoires Vivants France.

## 1. Parties

- Collectivite :
- Service referent :
- Representant :
- Territoires Vivants France :
- Referent TVF :

## 2. Objet

La convention vise a organiser une cooperation autour de la revitalisation territoriale :

- patrimoine vacant ;
- commerces fermes ;
- friches et terrains inutilises ;
- materiaux de reemploi ;
- mobilisation citoyenne ;
- insertion ;
- etudes territoriales.

## 3. Perimetre territorial

- Commune ou intercommunalite :
- Quartiers concernes :
- Publics concernes :
- Ressources identifiees :
- Limites du perimetre :

## 4. Contributions possibles de la collectivite

- Donnees publiques ou internes partageables ;
- Locaux ou espaces disponibles ;
- Materiaux, mobilier ou equipements inutilises ;
- Mise en relation avec acteurs locaux ;
- Appui logistique ;
- Participation a un comite de suivi ;
- Communication institutionnelle validee.

## 5. Contributions possibles de TVF

- Methode de diagnostic ;
- Structuration du signalement ;
- Cartographie preparatoire ;
- Animation territoriale ;
- Mobilisation de partenaires ;
- Preparation de fiches projets ;
- Suivi des indicateurs ;
- Documentation des enseignements.

## 6. Gouvernance

La convention doit prevoir :

- referents ;
- frequence des reunions ;
- modalites de decision ;
- circuit de validation des donnees ;
- regles de confidentialite ;
- modalites de publication.

## 7. Usages territoriaux

Lorsque des biens ou ressources publiques sont mobilises, la convention peut prevoir :

- activites pour les habitants ;
- accueil d'associations ;
- ateliers pedagogiques ;
- animations culturelles ;
- services a la population ;
- experimentation d'un usage temporaire.

## 8. Donnees et cartographie

Les donnees sensibles ne sont pas publiees automatiquement.  
Chaque publication doit preciser :

- source ;
- date ;
- territoire ;
- niveau de validation ;
- limites d'interpretation.

## 9. Suivi et bilan

Indicateurs possibles :

- biens signales ;
- biens qualifies ;
- ressources identifiees ;
- projets instruits ;
- acteurs mobilises ;
- livrables produits ;
- difficultes rencontrees.

## 10. Duree et fin

- Duree initiale :
- Conditions de renouvellement :
- Sortie anticipee :
- Bilan final :
- Archivage des donnees :

## 11. Communication

Aucune cooperation ne doit etre presentee comme acquise avant signature.  
Toute mention publique de la collectivite, de TVF ou d'un partenaire doit etre validee.
