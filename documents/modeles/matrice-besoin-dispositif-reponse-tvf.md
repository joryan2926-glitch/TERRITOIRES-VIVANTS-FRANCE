# Matrice - Besoin territorial, dispositif existant et reponse TVF

Statut : modele de travail TVF a adapter au territoire.
Objectif : montrer comment TVF peut completer les politiques publiques sans les remplacer.

## 1. Principe d'utilisation

Cette matrice sert a preparer une discussion avec une collectivite. Elle distingue :

- le besoin concret observe ;
- le dispositif public ou local deja mobilisable ;
- la difficulte de mise en oeuvre ;
- la contribution possible de TVF ;
- les partenaires a associer ;
- l'impact attendu ;
- les preuves a produire.

## 3. Grille de lecture pour Saint-Etienne

Pour Saint-Etienne, la matrice doit etre reliee aux sujets suivants lorsque les donnees et programmes sont confirmes :

- centre ancien et habitat degrade ;
- logements vacants et remise en usage ;
- revitalisation des rez-de-chaussee ;
- friches et recyclage foncier ;
- economie circulaire et reemploi ;
- renovation energetique ;
- insertion professionnelle ;
- participation citoyenne ;
- adaptation climatique et espaces de proximite.

## 4. Regle de presentation

Ne jamais presenter une ligne de la matrice comme un projet valide. Une ligne est seulement une piste d'instruction tant que :

- le bien ou la ressource n'est pas identifie ;
- le proprietaire n'a pas donne son accord ;
- la collectivite n'a pas valide le cadre ;
- le budget n'est pas documente ;
- les risques ne sont pas verifies ;
- la convention n'est pas signee.
