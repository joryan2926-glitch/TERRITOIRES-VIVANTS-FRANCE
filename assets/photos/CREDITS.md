# Credits photographiques

Les photographies sont utilisees comme illustrations documentaires. Elles ne presentent pas des projets realises par Territoires Vivants France, sauf mention explicite contraire.

## Banque d'images TVF - Version 1.0

La version 1.0 du site utilise des variantes web stockees dans `assets/photos/v1/`.
Ces variantes sont generees a partir des photographies francaises verifiees listees ci-dessous.
Chaque image HTML conserve un attribut `data-france-source` indiquant le fichier source.

Le registre complet des usages est disponible dans :

- `assets/photos/v1/BANQUE_IMAGES_V1.md`
- `assets/photos/v1/manifest.json`

Les variantes ne constituent pas de nouvelles realisations TVF : elles servent uniquement a donner une direction artistique coherente, francaise et professionnelle au site.
