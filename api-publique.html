API publique

Préparer les endpoints qui alimenteront la future application mobile et les services partenaires autorisés.

GET /signalements

Retourne les signalements validés. POST nécessite une session Supabase.

GET /materiaux

Retourne les matériaux validés et disponibles publiquement.

GET /projets

Retourne les projets actifs ou publiables.

GET /territoires

Retourne la liste des territoires configurés.

Sécurité

Les écritures nécessitent un jeton utilisateur. Les validations administrateur passent par `/api/admin` et un rôle `administrateur` dans `user_profiles`.
